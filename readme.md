# commands

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Compiles and minifies for local
```
yarn run buildDev
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Visual interface
```
vue ui
```

### 获取webpack配置
cmd 中敲 vue inspect > output.js , 会得到一份最终生效的 webpack 配置信息。
